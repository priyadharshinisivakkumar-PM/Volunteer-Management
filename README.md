# Volunteer-Management website
This Volunteer Management Product is a  
1. User friendly website for volunteers to register
2. Supports Temple organizers to manage the volunteers registeration
3. Allows Team registration of volunteers
4. Helps organizers to plan for the events with the number of volunteers registered for each event

Full-stack starter project for a volunteer management website:

- Frontend: React + Vite
- Backend: Node.js + Express + PostgreSQL + Supabase Storage

## Prerequisites

- Node.js 20+ (recommended for Supabase SDK compatibility)
